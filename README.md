A Python program to visualize and predict the sea levels.

## Technologies Used
- pandas
- matplotlib
- scipy stats

## Features
- Generating a scatter plot of the sea levels throughout the years
- Adding two linear regression lines to predict the future levels of the sea until 2050, one calculated with all the dates available and the other since 2000
